# Pong
Pong using SFML 🏓

This is a packaged copy of the Pong example distributed with the SFML source-code.

## Setup

Dependencies are managed via [Buckaroo](https://www.buckaroo.pm).

```
buckaroo install
```

## Run

Builds are handled via [Buck](https://buckbuild.com/). 

```
buck run :pong
```
